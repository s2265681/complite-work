Status

a —— git add .
space —— 取消暂存
c —— git commit 
A —— amend 提交到上次提交的里面  git commit --amend  --no-edit --allow-empty

q —— 关lazygit
lazygit —— 打开
d —— 删除该文件
D -- 更多删除选择
p -- pull
P -- push

branches面板

n _ 新建分支 在哪个分支使用就是基于哪个
s _ stash  暂存起来
g _ pop 出来
d _ drop 掉satsh的东西
M _ 合并分支 在当前接收的这个分支上  在要合并的分支上按M合并 
[ ] 切换tag或者分支
，。 翻页 
<  >  最上最下
/ 搜索
