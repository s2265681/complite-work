# 数据结构

数据结构：计算机的存储、组织的方式，就像灌碗瓢盆
算法： 一系列解决问题的清晰指令，就像食谱
程序 = 数据结构 + 算法
数据结构为算法提供服务，算法围绕数据结构操作

## 动态规划

暴力递归 fib

```js
function fib(n) {
  if (n == 1 || n == 2) return 1;
  return fib(n - 1) + fib(n - 2);
}
```
